# Fix Scripts - Remoti (Launcher)

Launcher remoti per script di fix e troubleshooting.

## Script Disponibili

- `rfix-frp-checkmk-host.sh` - Fix configurazione FRP host CheckMK
- `rforce-update-checkmk.sh` - Force update CheckMK (risolve problemi update)

## Uso

```bash
# Esegui direttamente
./rfix-frp-checkmk-host.sh

# O scarica ed esegui
curl -fsSL https://raw.githubusercontent.com/Coverup20/checkmk-tools/main/Fix/remote/rfix-frp-checkmk-host.sh | bash
```
