# Ydea Toolkit - Script Remoti (Launcher)

Launcher remoti per l'integrazione Ydea con CheckMK.

## Script Disponibili

- `rydea-toolkit.sh` - Toolkit completo Ydea
- `rydea-monitoring-integration.sh` - Integrazione monitoring
- `rydea-health-monitor.sh` - Health monitor
- `rydea-ticket-monitor.sh` - Monitor ticket
- `rydea-templates.sh` - Template ticket
- `rcreate-ticket-ita.sh` - Creazione ticket italiano
- `rtest-ydea-integration.sh` - Test integrazione
- `rinstall-ydea-checkmk-integration.sh` - Installer integrazione
- `resempi-ydea.sh` - Esempi utilizzo

## Uso

```bash
# Esegui direttamente
./rydea-toolkit.sh

# O scarica ed esegui
curl -fsSL https://raw.githubusercontent.com/Coverup20/checkmk-tools/main/Ydea-Toolkit/remote/rydea-toolkit.sh | bash
```
